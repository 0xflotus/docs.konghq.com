Docker

  1. Start Cassandra:

    docker run -p 9042:9042 -d --name cassandra mashape/docker-cassandra

  2. Start Kong:

    docker run -p 8000:8000 -p 8001:8001 -d --name kong --link cassandra:cassandra mashape/docker-kong:0.1.1beta-2

  3. Kong is running:

    curl http://127.0.0.1:8001

CentOS 6/7

  1. Installation:

    Add the following in your /etc/yum.repos.d/ directory in a file named (for example) kong.repo

    [kong]
name = Kong
baseurl = http://mashape-kong-yum-repo.s3-website-us-east-1.amazonaws.com/$releasever/$basearch
enabled = 1
gpgcheck = 0

    Then execute:

    yum install kong

  2. Start Kong:

    Before starting Kong, make sure Cassandra v2.1.3 is running and kong.yml points to the right Cassandra server. Then execute:

    kong start

  3. Kong is running:

    curl http://127.0.0.1:8001

Debian 7 Wheezy

  1. Installation:

    Add the right source to APT:

    echo "deb http://mashape-kong-apt-repo.s3-website-us-east-1.amazonaws.com/debian/wheezy/ kong main" >> /etc/apt/sources.list

    Then execute:

    apt-get update
apt-get install kong

  2. Start Kong:

    Before starting Kong, make sure Cassandra v2.1.3 is running and kong.yml points to the right Cassandra server. Then execute:

    kong start

  3. Kong is running:

    curl http://127.0.0.1:8001
