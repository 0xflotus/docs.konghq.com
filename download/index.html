Latest Kong version: 0.2.0-2 (see log history). Requires Apache Cassandra 2.1.2.

Docker

  1. Start Cassandra:

    docker run -p 9042:9042 -d --name cassandra mashape/docker-cassandra

  2. Start Kong:

    docker run -p 8000:8000 -p 8001:8001 -d --name kong --link cassandra:cassandra mashape/docker-kong:0.2.0-2

  3. Kong is running:

    curl http://127.0.0.1:8001

  4. Start using Kong:

    Quickly learn how to use Kong with the 5-minute Quickstart.

CentOS 6/7

  1. Installation:

    Add the following in your /etc/yum.repos.d/ directory in a file named (for example) kong.repo

    [kong]
name = Kong
baseurl = http://mashape-kong-yum-repo.s3-website-us-east-1.amazonaws.com/$releasever/$basearch
enabled = 1
gpgcheck = 0

    Then execute:

    yum install kong

  2. Start Kong:

    Before starting Kong, make sure Cassandra v2.1.2 is running and kong.yml points to the right Cassandra server. Then execute:

    kong start

  3. Kong is running:

    curl http://127.0.0.1:8001

  4. Start using Kong:

    Quickly learn how to use Kong with the 5-minute Quickstart.

Homebrew

Kong is available as a Homebrew recipe on GitHub: Mashape/homebrew-kong.

  1. Installation

    brew tap mashape/kong
brew install kong

    (Optional) If you want to use a local Cassandra cluster, this tap can also install the homebrew/cassandra formula if you run it with:

    brew tap mashape/kong
brew update # for the cassandra formula
brew install kong --with-cassandra

  2. Start Kong:

    Before starting Kong, make sure Cassandra v2.1.2 is running and kong.yml points to the right Cassandra server. Then execute:

    kong start

    You can install Cassandra with:

    brew install cassandra
# to start cassandra, just run `cassandra`

  3. Kong is running:

    curl http://127.0.0.1:8001

  4. Start using Kong:

    Quickly learn how to use Kong with the 5-minute Quickstart.

    Install Lua v5.1.5
