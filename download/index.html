Docker

Details about how to use Kong in Docker can be found on the Dockerhub repo hosting the image: mashape/kong.

Here is a quick setup linking Kong to a Cassandra container:

  1. Start Cassandra:

    $ docker run -p 9042:9042 -d --name cassandra mashape/cassandra

  2. Start Kong:

    $ docker run -p 8000:8000 -p 8001:8001 -d --name kong --link cassandra:cassandra mashape/kong

  3. Kong is running:

    $ curl http://127.0.0.1:8001

  4. Start using Kong:

    Quickly learn how to use Kong with the 5-minute Quickstart.

CentOS 5/6/7

  1. Installation:

    For CentOS 5/RHEL5 download this package: kong-0.2.1.el5.noarch.rpm - Recommended for Amazon Linux AMI

    For CentOS 6/RHEL6 download this package: kong-0.2.1.el6.noarch.rpm

    For CentOS 7/RHEL7 download this package: kong-0.2.1.el7.noarch.rpm

    Then execute:

    $ sudo yum install epel-release
$ sudo yum install kong-0.2.1.*.noarch.rpm --nogpgcheck

  2. Configure Cassandra

    Before starting Kong, make sure you have installed or provisioned Cassandra v2.1.5 and updated /etc/kong/kong.yml.

  3. Start Kong:

    $ kong start

# Kong is running
$ curl 127.0.0.1:8001

  4. Use Kong:

    Quickly learn how to use Kong with the 5-minute Quickstart.

Other

  1. Install dependencies:

    Install Lua v5.1.4

    Install Luarocks v2.2.2

    Install Dnsmasq

    Install OpenResty v1.7.10.2 (OpenResty has some dependencies of its own), with the following configure options:

    $ ./configure --with-pcre-jit --with-ipv6 --with-http_realip_module --with-http_ssl_module --with-http_stub_status_module

    Install OpenSSL

    Install PCRE

    Some of the dependencies may be available in your favorite package manager.

  2. Install Kong:

    $ luarocks install kong 0.2.1-1

    Or

    $ git clone git@github.com:Mashape/kong.git
$ [sudo] make install

  3. Configure Cassandra

    Before starting Kong, make sure you have installed or provisioned Cassandra v2.1.5 and updated /etc/kong/kong.yml.

  4. Start Kong:

    $ kong start

# Kong is running
$ curl 127.0.0.1:8001

  5. Use Kong:

    Quickly learn how to use Kong with the 5-minute Quickstart.
